Non-Semantic_Tags(div & span)

Div tag acts as a container for other tags and does not have its own meaning.

This is div tag and known as block element bcz it take full width

It can contain collection of different tags like; etc


Span tag as an inline container.

This is a span tag. It only takes up as much space as needed and is used for styling small parts of text. Common ways is used with: Classes (like .highlight, .error, etc.) IDs, Styling purposes, JavaScript hooks.
